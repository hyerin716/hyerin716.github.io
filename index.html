개요
- Spring Boot를 기반으로 MySQL과 MyBatis를 활용한 구현된 옷 쇼핑몰 웹사이트 제작.
- 사용자와 관리자 권한을 나눠서 페이지 제작

사용자가 원하는 상품을 장바구니에 담고 구매하며, 구매한 상품에 대해 리뷰를 남길 수 있는 옷 쇼핑몰 웹사이트.
관리자는 상품, 카테고리, 회원, 관리자 관리 기능을 제공하며, 주문이 들어오면 해당 상품의 배송 처리까지 원활하게 진행할 수 있도록 구현되었다.

담당기능

관리자 페이지
1. 카테고리 관리
  • 카테고리 리스트 출력 (페이징 처리)
  • 카테고리 추가 기능 구현
  • 카테고리 삭제 기능 구현
 2. 상품 관리
  • 상품 리스트 카드 형식으로 출력되도록 구현
  • 상품 상세 조회 기능 구현
  • 상품 등록 기능(파일 첨부 ,카테고리 선택, 유효성 검사 포함)
  • 상품 삭제 기능 구현
고객 페이지
1. 메인화면
  • 로그인 상태에 따른 회원/비회원 구분해서 페이지 구현
  • 카테고리별 상품 출력되도록 구현
  • 재고없는 경우 품절 표시되도록 구현
